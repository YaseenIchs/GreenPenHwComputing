HTML tags can be used to make a simple website?


For a simple website such as this you can use tags such these:


  • The body{} inside the style tag can allow thing such as this background image
  • The li tag can make bullet point lists like this
  • The h1-h6 tags can be good for making a heading such as the one under the ICHS logo
  • The a tag is useful in anchoring together 2 things such as this image which ,when clicked, takes you to the ICHS website
  • the img src tag adds images such as the ICHS logo
  • The u tag underlines any text within such as the heading
  • The b tag makes any text bold
  • The i tag makes things italics like the title at the top
